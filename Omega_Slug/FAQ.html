Who are we and why did we create this game?

We are 4 students from Uskudar University studying computer science, trying to
Develop a game for the sake of improving ourselves and finishing our software internship :)

How much time did you spend developing the game?

We spent approximately 1 month on the game. Including the webpage (Which is made by us).

Which programming language and engine
did you use for developing the project?

We used Unity Engine for the game development which uses C# as it's programming language.
The website was created using HTML/CSS/JS (and additional frameworks)
The assets in game were created using illustration
softwares such as Adobe Illustrator, Adobe Photoshop etc.

How much time did it cost you to develop the game?

It didn't cost much. 90% of the assets used in the game were created by us.
The bought ones were either very cheap or free. Game engine was also free because
of us being students.The only cost was the domain and the hosting for the website

Did you have fun while developing the project?

Yes! We had a lot of fun trying to solve the bugs and the errors together.
Although we had some troubles on the way trying to solve the problems,
we managed to get over them and finish the project successfully.
Overall this project was a very great,instructive and pleasurable experience
